# Guia de Contribuição

Obrigado por contribuir com o MCP Licitações!

## Como contribuir

1. Faça um fork do repositório
2. Crie uma branch para sua feature ou correção: `git checkout -b minha-feature`
3. Faça commits claros e objetivos
4. Rode os testes antes de enviar PR: `pytest tests/`
5. Abra um Pull Request detalhando sua contribuição

## Padrão de código

- Siga o PEP8 (flake8 já está configurado)
- Documente funções e classes com docstrings
- Adicione testes para novas funcionalidades

## Dúvidas?

Abra uma issue ou entre em contato!
